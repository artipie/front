To install packages from rpm Artipie repository, add the following file to yum settings by path /etc/yum.repos.d/example.repo: 
[example]
name=Example Repository
baseurl=http://{{user}}:password@central.artipie.com/{{user}}/{{repo}}
enabled=1
gpgcheck=0
Then list and install packages with specified repository: 
yum -y repo-pkgs example list
yum -y repo-pkgs example install
