To publish your packages with twine specify Artipie repository url with --repository-url option 
$ twine upload --repository-url http://central.artipie.com/{{user}}/{{repo}}/ -u {{user}} -p password myproject/dist/*
To install package with pip install specify Artipie repository url with --index-url option: 
$ pip install --index-url http://{{user}}:password@central.artipie.com/{{user}}/{{repo}}/ myproject
