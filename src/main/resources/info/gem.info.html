Before uploading your gems, obtain a key for authorization. A base64 encoded login:password would be a valid key: 
    export GEM_HOST_API_KEY=$(echo -n "{{user}}:****" | base64)
Now, to upload .gem file into Artipie repository, use gem push command: 
    gem push  --host https://central.artipie.com/{{user}}/{{repo}}
To install an existing gem, use gem install command: 
    gem install awesome_gem --source https://central.artipie.com/{{user}}/{{repo}}
