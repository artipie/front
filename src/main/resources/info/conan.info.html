To install conan packages from Artipie, add repository URL to conan using command below. Note that `False` is required to force disable SSL protocol. Check documentation for Conan 1.x for more details): 
  conan remote add conan-test http://artipie.artipie:9300 False
When you upload or download package you can specify remote repository explicitly in conan client by `-r` option. 
conan upload -r conan-test ...
conan download -r conan-test ...
